Breast Mass Detection using YOLOv11

Breast cancer is one of the most common cancers among women worldwide, and early detection plays a critical role in improving survival rates. This project focuses on developing an automated breast mass detection system using YOLOv11, a state-of-the-art real-time object detection model.

The model is trained on mammographic images to accurately identify and localize potential breast masses. By leveraging deep learning and computer vision techniques, the system aims to assist in faster and more reliable analysis of mammograms.

Key aspects of the project include:

Preprocessing and annotation of mammogram images

Training a YOLOv11 object detection model for mass localization

Evaluating model performance using standard detection metrics

Performing inference on new mammogram images for automated detection

This project demonstrates how deep learning and computer vision can be applied to medical imaging to support early breast cancer detection and improve diagnostic efficiency.
